# Задача 3.1
## То ли акростих, то ли акроним, то ли апроним
Акростих - осмысленный текст, сложенный из начальных букв каждой строки стихотворения. Акроним - вид абривиатуры, образованный начальными звуками (например НАТО, вуз, НАСА, ТАСС), которые можно произнести слитно (в отличии от аббривиатуры, которую произносят по буквам, например: "кэ-гэ-бэ"). 

На вход даётся текст. Выведете слитно первые буквы каждого слова. Буквы необходимо выводить заглавными. Эту задачку можно решить в одну строку.

```
Входные данные:
Московский государственный институт междунородных отношений

Выходные данные:
МГИМО

Входные данные:
микоян авиацию снабдил алкоголем,
народ доволен работой авиаконструктора

Выходные данные:
МАСАНДРА
```
